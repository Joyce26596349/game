# 🔢 1A2B 邏輯挑戰賽

這是使用 C++ 撰寫的經典邏輯遊戲。玩家需要根據系統回傳的 A（數字對且位置對）與 B（數字對但位置錯）來推敲出正確答案。

### 📜 遊戲規則
1. 輸入 4 個不重複的數字。
2. **A**：數字正確且位置正確。
3. **B**：數字正確但位置錯誤。
4. 輸入 `0000` 可結束遊戲，輸入 `1111` 開始新一輪。
